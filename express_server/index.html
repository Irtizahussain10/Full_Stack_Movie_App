This site can be used to make different queries for fetching data. The description is given below:

API Description
http://localhost:5000/ Description of all APIs available
http://localhost:5000/allMovies/3 Twenty movies present on third page (page number can be changed) and total movies in the collection
http://localhost:5000/searchMoviesByCast/John Ott Movies by this actor
http://localhost:5000/searchMoviesByGenre/Fantasy/4 Movies of this genre
http://localhost:5000/searchMoviesByCountry/USA/1 Movies released in this country
http://localhost:5000/searchMoviesById/573a1391f29313caabcd70b4 Movie against its id
